# LeagueCommunity

## 사용 NPM 종류
| Npm | 내용 |
|----|------------------|
|express| For Router |
|nodemon | For Encreasing Development |
|cookie-parser|For Using Cookie| 
|debug|For Printing Status|
|http-errors|For Handle To Error|
|dotenv|For Loading Environment Variables|
|mysql2|For Connect Mysql Server|
